# Demo Scenario

## 목표

카카오 Agentic Player 10 공모전에서 MCP 서버가 어떤 사용자 문제를 해결하는지 보여주는 데모 시나리오를 정리합니다.

## 흐름

1. 사용자가 에이전트에게 목표를 전달합니다.
2. 에이전트가 MCP 도구를 호출해 필요한 정보를 조회하거나 작업을 수행합니다.
3. MCP 서버가 구조화된 결과를 반환합니다.
4. 에이전트가 사용자에게 실행 가능한 다음 행동을 제안합니다.
